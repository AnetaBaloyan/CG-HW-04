In this project, I am implementing **ray casting** with _C++_ to draw triangles. 
* To test my results, I have created 3 different images shown below. You can find all the code in [`main.cpp`](https://github.com/AnetaBaloyan/CG-HW-04/blob/main/main.cpp). 
* To write the images to `.bmp` files, I am using the header files from [this source](https://github.com/nothings/stb/blob/master/stb_image_write.h).
